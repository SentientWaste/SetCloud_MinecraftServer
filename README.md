# 了解与介绍
`SetCloud`是一个`方便`，`快捷`，`实用`的开服器，支持下载服务端，宣传服务器，Java运行时安装，拥有集云电台，可以让您在使用途中放松心情。支持`高度DIY`，打造属于您的开服器。
